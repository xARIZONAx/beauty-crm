# BeautyRecall CRM

Statyczny CRM dla salonu beauty, gotowy do hostowania na GitHub Pages.

## Funkcje

- baza klientów z tagami, preferencjami kontaktu i notatkami,
- terminarz wizyt z kwotami i statusami,
- przypomnienia follow-up po wizycie,
- gotowe szablony wiadomości do kampanii,
- klikalne segmenty kampanii z listą klientów,
- bezpośrednie akcje kontaktu w przypomnieniach: WhatsApp, SMS, telefon, e-mail i Instagram,
- globalne wyszukiwanie,
- eksport i import danych JSON,
- zapis danych w `localStorage`.

## Uruchomienie lokalne

Otwórz `index.html` w przeglądarce albo uruchom prosty serwer statyczny:

```bash
python -m http.server 4173
```

Następnie wejdź na `http://127.0.0.1:4173/`.
